Your Healthcare Agent

Set up your AI health agent in 10 minutes.

Your Healthcare Agent is an open-source AI agent that helps you — or someone you care for — understand insurance coverage, navigate billing and claims, exercise appeal rights, and get the care you need. You bring your situation. The agent brings the expertise.

The whole agent is a single block of instructions you paste into Claude. The code, the spec, and the setup guide are open source.

Set up the agent

  1. Open Claude.ai → Projects and create a new project.
  2. Open Project Instructions.
  3. Copy the block below and paste it in.
  4. Save. You’re done.
